# Personal Profile & Keiba Dashboard

川田将雅騎手のようなプロフェッショナルで高級感のある競馬収支公開サイト

## 🎯 プロジェクトコンセプト

「ストイック」「高級感」「プロフェッショナル」な印象を与える個人プロフィール兼、競馬収支公開サイト。
競馬を「単なるギャンブル」ではなく「研ぎ澄まされた投資・大人の趣味」として表現。

## 🎨 デザイン特徴

- **カラーパレット**: ピュアブラック、ホワイト、シャンパンゴールド、シルバー
- **タイポグラフィ**: 高級感のある明朝体とサンセリフ体の組み合わせ
- **アニメーション**: スクロールに応じた要素の浮き上がり、ホバーエフェクト
- **レスポンシブ**: モバイル・タブレット・デスクトップ対応

## 🚀 使い方

### ローカルサーバーで実行（推奨）

CSVファイルを正しく読み込むには、ローカルサーバーが必要です。

#### 方法1: Python（簡単）
```bash
# Python 3がインストールされている場合
cd keiba-dashboard
python -m http.server 8000
```
ブラウザで `http://localhost:8000` を開く

#### 方法2: VS Code Live Server
1. VS Codeで `keiba-dashboard` フォルダを開く
2. Live Server拡張機能をインストール
3. `index.html` を右クリック → "Open with Live Server"

#### 方法3: Node.js（Node.jsがインストールされている場合）
```bash
npx serve
```

### データ更新方法

**重要**: コードを一切触らずに、CSVファイルの差し替えのみで更新可能!

1. `public_test/data/累月.csv` を編集
2. 以下の形式でデータを追加:

```csv
日付,レース名,投資額,払戻金,収支,回収率
2026/01/05,中山金杯,10000,15000,5000,150%
```

3. ファイルを保存
4. ブラウザをリロード → 自動的に統計・グラフ・テーブルが更新される!
